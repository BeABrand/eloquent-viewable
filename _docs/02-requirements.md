The Eloquent Viewable requires **PHP 7+** and **Laravel 5.5+**.

| Version | Illuminate    | Status         | PHP Version |
|---------|---------------|----------------|-------------|
| 2.0     | 5.5 - 5.6     | Active support | >= 7.0.0    |
| 1.0     | 5.5 - 5.6     | Bug fixes only | >= 7.0.0    |
